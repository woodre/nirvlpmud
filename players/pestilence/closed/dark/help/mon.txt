              S E T M O N
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Usage:	setmon 

This is a normal game command with some
guild enhancements added.  When using
setmon if you use #g1# it will display
your spell delay, #g2# will display if
you are in a pk zone.  #g3# will let
you know if enrage is active and #g4#
will let you know when enrage is available
